Chemin d'accès aux outils de génération. Il s'agit du répertoire dans lequel sont installés les outils de génération RTC. Exemple : C:\Program Files (x86)\IBM\TeamConcertBuild\buildsystem\buildtoolkit

Le chemin d'accès mène aux outils de génération sur le noeud maître. Si vous disposez d'une configuration de génération distribuée (agents), vous devez également spécifier l'emplacement des outils de génération sur l'agent (en supposant qu'ils se trouvent dans un autre chemin). Après la sauvegarde, accédez à la configuration de l'agent. Dans la section des propriétés de noeud, ajoutez des emplacements d'outils pour chacun des outils de génération à utiliser sur l'agent. Pour plus de détails, voir Configuration de noeud maître/agent.

Si vous prévoyez de charger des liens symboliques, des étapes de configuration supplémentaires sont nécessaires. L'accès aux bibliothèques natives est requis pour la création de liens symboliques. Pour plus de détails sur la configuration, voir Team Concert Plugin
