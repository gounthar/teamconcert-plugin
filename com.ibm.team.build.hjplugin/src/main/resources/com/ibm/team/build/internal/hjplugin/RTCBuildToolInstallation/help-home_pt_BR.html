O caminho para o Kit de Ferramentas de Compilação. Este será um diretório dentro das ferramentas de compilação do RTC instalado. Exemplo: C:\Program Files (x86)\IBM\TeamConcertBuild\buildsystem\buildtoolkit

O caminho é para o Kit de Ferramentas de Compilação no Principal. Se você tiver uma configuração de construção distribuída (agentes), também será necessário especificar onde o Build Toolkit está no agente (assumindo que ele está em um caminho diferente). Após salvar, acesse a configuração do agente. Na seção Propriedades do Nó, inclua os locais da Ferramenta para cada um dos kits de Ferramentas de Construção a serem usados no agente Consulte Configuração do Principal / Agente para obter detalhes.

Se você pretende carregar links simbólicos, é necessário fazer uma configuração adicional. O acesso às bibliotecas nativas é necessário para a criação de links simbólicos. Para obter mais detalhes sobre a configuração, visite Team Concert Plugin
