Pfad zum Build Toolkit. Dies ist ein Verzeichnis in der Installation der RTC Build Tools. Beispiel: C:\Program Files (x86)\IBM\TeamConcertBuild\buildsystem\buildtoolkit

Der Pfad zum Build-Toolkit auf dem Master. Bei einer verteilten Buildkonfiguration (Agenten) müssen Sie auch angeben, wo sich das Build Toolkit auf dem Agenten befindet (vorausgesetzt, es befindet sich in einem anderen Pfad). Wechseln Sie nach dem Speichern zur Konfiguration des Agenten. Fügen Sie im Abschnitt "Knoteneigenschaften" Toolpositionen für jedes der Build-Toolkits hinzu, die auf dem Agenten verwendet werden sollen. Weitere Informationen finden Sie unter Master-/Agentenkonfiguration.

Wenn Sie beabsichtigen, symbolische Links zu laden, müssen zusätzliche Konfigurationsschritte ausgeführt werden. Um symbolische Links zu erstellen, ist ein Zugriff auf native Bibliotheken erforderlich. Weitere Informationen zur Konfiguration finden Sie unter Team Concert Plugin.
