Demander une génération

Demandez une génération EWM (RTC) à partir du pipeline Jenkins. Lorsqu'une demande est créée, l'étape renvoie l'UUID du résultat de génération dans la propriété « buildResultUUID ».


Attendre la génération

Attendez qu'un résultat de génération EWM (RTC) atteigne un état spécifique. Les états par défaut à attendre sont COMPLETED et INCOMPLETE. Si un délai d'attente est indiqué, la tâche n'est pas renvoyée tant que l'état spécifié n'est pas atteint ou que le délai n'est pas expiré. Par défaut, la tâche attend indéfiniment jusqu'à ce que le résultat de génération atteigne les états spécifiés.


Répertorier les journaux

Extrayez les détails d'un ou de plusieurs journaux d'un résultat de génération EWM (RTC). Les journaux apparaissent dans l'onglet « Journaux » du résultat de génération EWM. Cette tâche extrait les détails des journaux dont le contenu est stocké dans le référentiel EWM.


Répertorier les artefacts

Extrayez les détails des artefacts d'un résultat de génération EWM (RTC). Les artefacts apparaissent dans l'onglet « Téléchargements » du résultat de génération EWM. Cette tâche extrait les détails des artefacts dont le contenu est stocké dans le référentiel EWM.


Journal de téléchargement

Téléchargez un journal à partir d'un résultat de génération EWM (RTC). Vous pouvez télécharger un journal à l'aide du nom du journal ou de l'ID de contenu. Pour extraire le nom ou contentId du journal, utilisez la tâche « Répertorier les journaux ».


Télécharger l'artefact

Téléchargez un artefact à partir d'un résultat de génération EWM (RTC). Vous pouvez télécharger un artefact à l'aide du nom de l'artefact ou de l'ID de contenu. Pour extraire le nom ou contentId de l'artefact, utilisez la tâche « Répertorier les artefacts ».


Obtenir l'image instantanée SCM à partir de la génération

Extrayez les détails de l'instantané Jazz SCM pour un résultat de génération EWM (RTC) donné. Si le résultat de la génération comporte une contribution d'image instantanée Jazz SCM, la tâche renvoie le nom et l'UUID de l'image instantanée. Si le résultat de génération n'a pas de contribution de ce type, la tâche renvoie des valeurs vides pour le nom de l'image instantanée et l'identificateur unique universel.
