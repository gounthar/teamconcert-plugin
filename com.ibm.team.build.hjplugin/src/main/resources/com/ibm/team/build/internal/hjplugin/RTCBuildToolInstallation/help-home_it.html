Il percorso del toolkit della build. Questa sarà una directory compresa nei tool di build RTC installati. Esempio: C:\Program Files (x86)\IBM\TeamConcertBuild\buildsystem\buildtoolkit

Il percorso è quello del toolkit della build in quello principale. Se si dispone di una configurazione di build distribuita (agent), sarà necessario specificare anche dove Build Toolkit si trova sull'agent (supponendo che si trovi in un percorso diverso). Dopo il salvataggio, passare alla configurazione dell'agente. Nella sezione Proprietà nodo, aggiungere le ubicazioni degli strumenti per ciascuno dei toolkit di build da utilizzare sull'agent. Consultare Configurazione master/agent per i dettagli.

Se si intende caricare link simbolici, è necessario eseguire alcune configurazioni aggiuntive. L'accesso alle librerie native è richiesto per creare link simbolici. Per ulteriori dettagli sulla configurazione, visitare Team Concert Plugin
