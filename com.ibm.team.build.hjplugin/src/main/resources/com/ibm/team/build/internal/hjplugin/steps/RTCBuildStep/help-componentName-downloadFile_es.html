Los registros y artefactos se pueden agrupar bajo un componente en el resultado de compilación. Al proporcionar un nombre de componente, puede identificar de forma exclusiva un registro o artefacto, especialmente cuando hay varios registros o artefactos con el mismo nombre de archivo en distintos componentes pero no dentro del mismo componente.

Si hay varios registros o artefactos con el mismo nombre de archivo bajo un componente, utilice el campo ID de contenido para identificar de forma exclusiva la contribución.

Si deja este campo en blanco, se recuperará el primer registro o artefacto que coincida con el nombre de archivo.
