Eseguire la consegna post-build per la configurazione SCM RTC (Rational Team Concert) nella build Jenkins.

In un processo pipeline in cui RTC SCM può essere richiamato più volte, la consegna post-build verrà tentata per ogni richiamo di RTC SCM.

Attualmente, la consegna post-build viene eseguita solo se viene utilizzata una definizione di build come configurazione di build. La fase utilizza le informazioni di configurazione della consegna post-build dalla definizione della build.
