Les journaux et les artefacts peuvent être regroupés sous un composant dans le résultat de génération. En fournissant un nom de composant, vous pouvez identifier de manière unique un journal ou un artefact, en particulier lorsqu'il existe plusieurs journaux ou artefacts portant le même nom de fichier dans différents composants, mais pas dans le même composant.

S'il existe plusieurs journaux ou artefacts avec le même nom de fichier sous un composant, utilisez la zone ID de contenu pour identifier de manière unique la contribution.

Si vous laissez cette zone vide, le premier journal ou artefact correspondant au nom de fichier sera extrait.
