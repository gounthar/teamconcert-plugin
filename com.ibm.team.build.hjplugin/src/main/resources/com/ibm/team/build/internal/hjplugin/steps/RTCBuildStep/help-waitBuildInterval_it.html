Il numero in secondi tra due query successive al server EWM per lo stato di build durante l'attesa della build. Il valore predefinito è 30 secondi. Il valore minimo è 1 e il valore massimo può essere un qualsiasi numero intero positivo inferiore al valore di timeout di attesa della build. Se il timeout di attesa della build è -1, il valore massimo deve essere inferiore a Long.MAX_VALUE.
