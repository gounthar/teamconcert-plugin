Effectuez la distribution post-génération pour la configuration Rational Team Concert(RTC) SCM dans la génération Jenkins.

Dans un travail de pipeline où RTC SCM peut être appelé plusieurs fois, la distribution post-génération sera tentée pour chaque appel de RTC SCM.

Actuellement, la distribution post-génération n'est effectuée que si une définition de génération est utilisée comme configuration de génération. L'étape utilise les informations de configuration de distribution post-génération de la définition de génération.
