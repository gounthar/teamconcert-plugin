Build anfordern

Fordern Sie einen EWM-Build (RTC) aus der Jenkins-Pipeline an. Wenn eine Anforderung erfolgreich erstellt wurde, gibt der Schritt die UUID des Buildergebnisses in der Eigenschaft "buildResultUUID" zurück.


Auf Build warten

Warten Sie, bis das Ergebnis für ein EWM-Build (RTC) einen bestimmten Status erreicht. Die Standardstatus, auf die gewartet werden soll, sind COMPLETED und INCOMPLETE. Wenn ein Zeitlimitintervall angegeben ist, kehrt die Task erst zurück, wenn der angegebene Status erreicht oder der Zeitraum abgelaufen ist. Standardmäßig wartet die Task unbegrenzt, bis das Buildergebnis die angegebenen Status erreicht.


Protokolle auflisten

Rufen Sie die Details eines oder mehrerer Protokolle aus dem Ergebnis für einen EWM-Build (RTC) ab. Protokolle werden auf der Registerkarte "Protokolle" des EWM-Buildergebnisses angezeigt. Diese Task ruft die Details der Protokolle ab, deren Inhalt im EWM-Repository gespeichert ist.


Artefakte auflisten

Rufen Sie die Details der Artefakte dem Ergebnis für einen EWM-Build (RTC) ab. Artefakte werden auf der Registerkarte "Downloads" des EWM-Buildergebnisses angezeigt. Diese Task ruft die Details der Artefakte ab, deren Inhalt im EWM-Repository gespeichert ist.


Protokoll herunterladen

Laden Sie ein Protokoll aus dem Ergebnis für einen EWM-Build (RTC) herunter. Sie können ein Protokoll unter Verwendung des Protokollnamens oder der Inhalts-ID herunterladen. Verwenden Sie die Task "Protokolle auflisten", um den Namen oder die contentId des Protokolls abzurufen.


Artefakt herunterladen

Laden Sie ein Artefakt aus dem Ergebnis eines EWM-Builds (RTC) herunter. Sie können ein Artefakt mit dem Namen oder der Inhalts-ID des Artefakts herunterladen. Verwenden Sie zum Abrufen des Namens oder der contentId des Artefakts die Task "Artefakte auflisten".


SCM-Momentaufnahme aus Build abrufen

Rufen Sie die Details der Jazz-SCM-Momentaufnahme für ein bestimmtes Ergebnis des EWM-Builds (RTC) ab. Wenn das Buildergebnis eine Ergänzung zur Jazz-SCM-Momentaufnahme enthält, gibt die Task den Namen und die UUID der Momentaufnahme zurück. Wenn das Buildergebnis keine solche Ergänzung enthält, gibt die Task leere Werte für den Namen der Momentaufnahme und die UUID zurück.
