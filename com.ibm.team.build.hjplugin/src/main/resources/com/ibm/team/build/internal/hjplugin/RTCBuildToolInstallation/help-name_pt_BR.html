O suporte do RTC SCM requer acesso a um kit de ferramentas de compilação no Nó em que uma compilação é executada. É possível configurar diversas versões de kits de ferramentas de compilação (exemplo: uma para RTC 4.0.2 e uma para RTC 3.0.1.5). Cada instalação de construção precisa receber um nome para ajudá-lo a selecionar o kit de ferramentas desejado ao definir as configurações globais, tarefas e nós do agente.
