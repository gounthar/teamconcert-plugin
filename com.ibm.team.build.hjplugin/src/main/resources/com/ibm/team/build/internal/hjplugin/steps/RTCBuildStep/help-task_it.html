Richiedi build

Richiedere una build EWM (RTC) dalla pipeline Jenkins . Quando una richiesta viene creata correttamente, il passo restituisce il risultato della build UUID nella proprietà "UUIDbuildResult".


Attendere la build

Attendere un risultato della build EWM (RTC) per ottenere uno stato specifico. Gli stati predefiniti da attendere sono COMPLETATO e INCOMPLETO. Se viene fornito un periodo di timeout, l'attività non viene restituita fino al raggiungimento dello stato specificato o fino alla scadenza del periodo di tempo. Per impostazione predefinita, l'attività attenderà indefinitamente fino a quando il risultato della build non raggiungerà gli stati specificati.


Elenca log

Richiamare i dettagli di uno o più log da un risultato di build RTC (EWM). I log vengono visualizzati nella scheda "Log" del risultato di build EWM. Questa attività richiamerà i dettagli dei log il cui contenuto è memorizzato nel repository EWM.


Elenca risorse utente

Richiamare i dettagli delle risorse utente da un risultato di build RTC (EWM). Le risorse utente vengono visualizzate nella scheda "Scaricamento" del risultato build EWM. Questa attività richiamerà i dettagli delle risorse utente il cui contenuto è memorizzato nel repository EWM.


Scarica log

Scaricare un log da un risultato di build RTC (EWM). È possibile scaricare un log utilizzando il nome del log o l'ID contenuto. Per richiamare il nome o contentId del log, utilizzare l'attività "Elenca log".


Scarica risorsa utente

Scaricare una risorsa utente da un risultato della build RTC (EWM). È possibile scaricare una risorsa utente utilizzando il nome della risorsa utente o l'ID contenuto. Per richiamare il nome o contentId della risorsa utente, utilizzare l'attività "Elenca risorse utente".


Acquisisci istantanea SCM da build

Richiamare i dettagli dell'istantanea Jazz SCM per un determinato risultato di build RTC (EWM). Se il risultato della build ha un contributo dell'istantanea Jazz SCM, l'attività restituisce il nome dell'istantanea e l'UUID. Se il risultato della build non dispone di tale contributo, l'attività restituisce valori vuoti per il nome istantanea e l'UUID.
