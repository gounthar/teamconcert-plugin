Für die RTC-SCM-Unterstützung muss auf ein Build Toolkit zugegriffen werden können, das sich auf dem Knoten befindet, auf dem Builds erstellt werden. Sie können mehrere Versionen des Build Toolkit konfigurieren (z. B ein Toolkit für RTC 4.0.2 und eines für RTC 3.0.1.5). Jede Buildinstallation muss einen Namen erhalten, damit Sie das gewünschte Toolkit auswählen können, wenn Sie die globalen Einstellungen, Jobs und Agentenknoten konfigurieren.
